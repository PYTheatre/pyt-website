## TROUBLESHOOTING CLOUDFLARE DEPLOYS (reference — I learned this today)

To check a deploy: dash.cloudflare.com → Workers & Pages → pyt-website → Deployments tab. Top row = newest. Green tick = success; yellow = building; red triangle = failed. For a failure, click "Details" on that row → Build log tab → scroll to the bottom red error (use "Copy log"). A failed build keeps the previous good version live, so new changes won't appear until the build passes.
